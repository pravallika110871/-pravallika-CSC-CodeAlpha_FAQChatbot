# Introduction

Chatbots are AI-based applications designed to communicate with users and provide instant responses. This project focuses on developing a FAQ chatbot that answers user questions based on predefined frequently asked questions.

The chatbot uses Natural Language Processing (NLP) techniques to identify similar questions and provide appropriate answers.

---

* To provide automatic responses to user queries.
* To understand NLP concepts and chatbot development.

# Methodology

1. Collect FAQ questions and answers.
2. Preprocess text using NLP techniques.
3. Convert text into vectors using TF-IDF.
4. Compare user input with stored FAQs.
5. Display the best matching answer.
